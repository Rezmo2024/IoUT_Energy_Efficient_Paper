# IoUT Energy Efficient Paper


** DOI: 10.5072/zenodo.565553 **

This repository contains the source code and implementation related to the research paper:

**"Optimizing Energy Consumption to Prolong the Lifetime of SDN-based IoUT Networks"**

The main objective of this work is to optimize energy consumption in Software-Defined Networking (SDN)-based Internet of Underwater Things (IoUT) networks in order to increase network lifetime and improve overall performance and efficiency.

## Requirements

Before running the project, make sure the following requirements are installed on your system:

- Python version **3.0 or above**
- Operating System:
  - Linux
  - Windows

You can verify your Python installation by running:

```bash
python3 --version
```

or on Windows:

```bash
python --version
```

## Installation

Clone the repository to your local machine using:

```bash
git clone https://github.com/Rezmo2024/IoUT_Energy_Efficient_Paper.git
```

Navigate to the project directory:

```bash
cd IoUT_Energy_Efficient_Paper
```
## Create the Resutls folder
```bash
mkdir ResultsFigs
```
```bash
mkdir ResultsText
```
```bash
mkdir Throughput
```



## Running the Project

To run the code, execute:

```bash
python3 Run.py
```

For some Windows systems where `python3` is not available, use:

```bash
python Run.py
```

## Optional Dependencies

If the project requires additional Python packages and a `requirements.txt` file is available, install the dependencies using:

```bash
pip install -r requirements.txt
```
